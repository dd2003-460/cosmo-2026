# 宇宙学详细讲解 — 模块3: 原初扰动功率谱（考试重点！）

## 模块定位

这是整个课程的**重中之重**。老师明确说考试会考功率谱的计算。

**整个逻辑链**:
```
暴胀期间存在量子涨落(无处不在)
    → 标量扰动: 密度/曲率扰动 → 结构形成的种子
    → 张量扰动: 引力波 → 原初引力波信号
    → 量子涨落在horizon crossing时被拉伸 → 冻结为经典扰动
    → 功率谱 P(k) = |δ_k|² 描述扰动的强度
    → n_s = d ln P/d ln k 描述谱的倾斜
    → 与 CMB 观测比较 → 约束暴胀模型
```

**关键洞察**: 今天我们看到的大尺度结构（星系、星系团），其种子是暴胀期间的**量子涨落**。这是量子力学和宇宙学的深刻联系。

## 1. 为什么需要扰动理论？

### 物理直觉

**问题**: FLRW 度规描述的是完全均匀的宇宙。但现实中宇宙不均匀 — 有星系、星系团。这些不均匀从哪来？

**思考路径**:

**Step 1**: 微扰 FLRW 度规
```
在 FLRW 背景上加小扰动:

g_μν = g_μν^FLRW + δg_μν

扰动可以分为三类(按变换性质):
- 标量扰动 δφ, δg_00, δg_0i, δg_ij^scalar → 密度扰动
- 矢量扰动 δg_0i^vector → 涡流(暴胀期间衰减)
- 张量扰动 δg_ij^tensor → 引力波

为什么只关心标量和张量?
- 矢量扰动在宇宙膨胀中衰减 → 不重要
- 标量扰动 → 密度扰动 → 结构形成
- 张量扰动 → 引力波 → CMB B模式极化
```

**动机**: 从对称性分类。标量、矢量、张量在空间旋转下的行为不同。标量最"普通"，首先考虑。

**Step 2**: 为什么量子涨落是关键？
```
经典理论: 初始条件需要人为设定（为什么？多出来的不确定性）

量子理论: 暴胀子场 φ 是量子场 → 有固有的量子涨落

量子场论告诉我们: 任何场都有不确定性:
δφ ≠ 0 (Huyzensberg uncertainty)

在暴胀期间，这些涨落被拉伸到宏观尺度 → 成为经典扰动 ✓
```

**动机**: 这是量子力学的基本原理。不需要额外假设。暴胀的独特之处在于把量子涨落放大到宇宙学尺度。

## 2. 标量扰动功率谱的计算

### Mukhanov-Sasaki 变量 — 怎么想到的？

**核心问题**: 如何从数学上处理标量扰动？

**思考路径**:

**Step 1**: 规范不变变量
```
扰动理论中最大的麻烦: 坐标选择会影响扰动的定义。

比如 "密度扰动 δρ" 在不同坐标系中值不同。

解决: 构造规范不变的组合。

Mukhanov-Sasaki 变量:
v = a × (δφ + φ̇ × ψ/H)

其中 ψ 是度规标量扰动。

为什么构造这个? 因为 v 的运动方程是一个谐振子！

v'' - ∇²v - (z''/z)v = 0

其中 ' = d/dη (共形时间), z = aφ̇/H
```

**动机**: 这不是"想到的"，而是"试出来的"。物理学家尝试各种组合，发现 v 的运动方程最简单。关键是记住: **扰动等效于一个质量随时间变化的谐振子**。

**Step 2**: 初始条件 — Bunch-Davies 真空
```
当扰动波长 << Hubble 半径时 (k >> aH):
时空近似平直 → 量子场处于真空态。

v_k(η) → (1/√(2k)) e^{-ikη} / √(2k)  (当 η → 0⁻)

这叫做 Bunch-Davies 真空。

物理含义: 在极早期，扰动处于最小不确定态。没有粒子，没有扰动。
```

**动机**: 这是量子场论在弯曲时空中最基本的假设。类比: 谐振子基态波函数是高斯型。

**Step 3**: horizon crossing 时的冻结

```
运动方程: v_k'' + (k² - z''/z)v_k = 0

当 k² >> z''/z (亚 horizon): 振荡解 e^{-ikη}
当 k² << z''/z (超 horizon): 冻结解 v_k = const

为什么冻结?

类比: 一个阻尼谐振子，当阻尼极大时，振荡停止，位移冻结。
在暴胀期间，Hubble 摩擦极大 → 扰动"冻结"在 horizon crossing 时的振幅。

具体计算:
在 horizon crossing (k = aH):
|δφ_k| ≈ H/(2π)  (量子涨落的 standard deviation)

为什么是 H/(2π)?

从量子场论: 每个模的零点涨落 = √(k/2) → 在共形时间中
转换到物理场: δφ_k = δχ_k/a → 在 horizon crossing 时
|δφ_k|² = H²/(2k³) → P_δφ(k) = k³/(2π²) × H²/(2k³) = H²/(4π²)
```

**动机**: H/(2π) 是暴胀宇宙学中最常见的数字之一。记住它。

**Step 4**: 从 δφ 到曲率扰动 ℛ

```
密度扰动本身不够 — 需要曲率扰动，因为度规扰动决定 CMB 温度涨落。

曲率扰动 (Maldacena 定义):
ℛ = ψ + H × δφ/φ̇

其中 ψ 是度规的空间标量扰动。

关键性质: 在超 horizon 尺度，ℛ 守恒 (不随时间变化)。

这意味着: 暴胀期间计算的 ℛ，可以直接用于今天的 CMB 观测。
```

**动机**: ℛ 的守恒性是 Maldacena (2003) 的重要发现。它是暴胀预言和观测之间的桥梁。

**Step 5**: 功率谱

```
定义功率谱:
⟨ℛ(k)ℛ(k')⟩ = (2π)³δ³(k+k') × P_ℛ(k)/2k³

计算:
P_ℛ(k) = (H²/(2πφ̇))² × H²/(2k³) × k³/(2π²)

= H²/(8π²φ̇²) × H²|_{k=aH}

用慢滚近似 φ̇ ≈ -V'/(3H):

φ̇² = V'²/(9H²)

P_ℛ = H²/(8π²) × 9H²/V'² × H²

= H²/(8π²M_pl²ε_V) × (1/M_pl² factor handled)

最终:
P_ℛ(k) = H²/(8π²M_pl²ε_V) |_{k=aH}
```

## 3. 谱指数 n_s

### 从功率谱到可观测

**思考路径**:

**Step 1**: 功率谱不是常数
```
P_ℛ(k) ∝ H²/ε_V

H 和 ε_V 都是 φ 的函数，而 φ 又和 k 相关(因为 k = aH 定义了 horizon crossing 时的 φ 值)。

所以 P_ℛ 是 k 的函数 → 谱指数 n_s ≠ 1。
```

**动机**: 如果 n_s = 1 (Harrison-Zel'dovich 谱)，意味着所有尺度的扰动强度相同。但暴胀子势能有斜率 → 不同时刻 horizon crossing 的 φ 值不同 → H 和 ε_V 不同 → n_s ≠ 1。

**Step 2**: 计算 n_s
```
n_s - 1 = d ln P_ℛ/d ln k

关键: d/d ln k = (dφ/d ln k) × d/dφ

从 horizon crossing 条件 k = aH:
d ln k = d ln a + d ln H = H dt + d ln H

dφ/d ln k = φ̇/H = -M_pl²V'/ (3H²) ≈ -M_pl²V'/V (慢滚)

所以:
d/d ln k ≈ (-M_pl²V'/V) d/dφ

现在:
ln P_ℛ = ln(H²/(8π²M_pl²ε_V))
= 2lnH - ln(8π²M_pl²) - lnε_V

d ln P_ℛ/d ln k = (-M_pl²V'/V) × d/dφ [2lnH - lnε_V]

用链式法则:
d lnH/dφ = H'/H = (1/2H²) × 2H'H = ... 

简化: 直接用慢滚参数表达:

n_s - 1 = 2η_V - 6ε_V    (谱指数公式)
```

**动机**: 这个计算是纯代数操作。关键是记住最终结果的形式:
- η_V 项来自 H 的变化 (动能项)
- ε_V 项来自 ε_V 自身的变化 (势能斜率)
- 系数 2 和 6 来自具体的代数运算

**Step 3**: 物理含义
```
n_s < 1 (红倾斜): 大尺度扰动更强

为什么? 因为在暴胀后期(φ 更小)，V' 更小 → ε_V 更小 → P_ℛ 更大。

但暴胀后期 horizon crossing 的模对应更大的尺度(更小的 k)。

所以: 小 k (大尺度) → P_ℛ 更大 → n_s < 1 ✓

观测: n_s = 0.965 ± 0.004 (Planck 2018)
→ 确实小于 1 → 暴胀预言被验证 ✓
```

## 4. 张量扰动 (原初引力波)

### 和标量扰动的对比

**思考路径**:

**Step 1**: 引力波是度规的张量扰动
```
h_ij 是横波无迹的张量场。

运动方程: h_k'' + 2ah' + k²h = 0

变换: μ = ah → μ_k'' + (k² - a''/a)μ_k = 0

这和标量扰动的方程形式相同！唯一的区别是"有效质量"项 a''/a 不同。
```

**动机**: 引力波也是量子场 → 也有零点涨落 → 也被暴胀拉伸 → 形成经典扰动。

**Step 2**: 张量功率谱
```
和标量扰动类似:
P_T(k) = 2 × H²/(π²M_pl²) |_{k=aH}

为什么有因子 2? 因为引力波有两个极化态 (h_+, h_×)。

因子 π² vs 8π²: 因为引力波是张量，标量是标量 → 不同的归一化。
```

### 张标比 r

```
r = P_T/P_ℛ = [2H²/(π²M_pl²)] / [H²/(8π²M_pl²ε_V)]

= 2/(π²M_pl²) × 8π²M_pl²ε_V

= 16ε_V

这个关系叫 consistency relation。

为什么重要?
- r 直接测量暴胀能标: V^{1/4} ∝ r^{1/4} × 10¹⁶ GeV
- 如果测到 r，就确定了暴胀的物理能标
- 如果 r = 0，意味着 ε_V = 0 → de Sitter → 没有结构形成

观测: r < 0.036 (Planck+BICEP/Keck 2018)
→ ε_V < 0.00225 → 慢滚条件确实满足 ✓
```

## 5. 典型计算题（考试类型）

### 题1: 给定势能，计算 n_s 和 r

**例**: V(φ) = ½m²φ²

```
Step 1: 计算慢滚参数
V'/V = m²φ/(½m²φ²) = 2/φ
V''/V = m²/(½m²φ²) = 2/φ²

ε_V = M_pl²/2 × (2/φ)² = 2M_pl²/φ²
η_V = M_pl² × 2/φ² = 2M_pl²/φ²

Step 2: 用 N 表示 φ
N ≈ ∫(V/V')dφ/M_pl² = ∫(φ/2)dφ/M_pl² = φ²/(4M_pl²)

→ φ² = 4NM_pl²

Step 3: 代入
ε_V = 2/(4N) = 1/(2N)
η_V = 2/(4N) = 1/(2N)

n_s - 1 = 2η_V - 6ε_V = 1/N - 3/N = -2/N
r = 16ε_V = 8/N

当 N = 60:
n_s = 1 - 2/60 ≈ 0.967
r = 8/60 ≈ 0.133
```

### 题2: 给定 n_s，判断模型是否被排除

```
φ⁴ 势: n_s = 1 - 6/(N+2)
当 N = 60: n_s ≈ 0.902 → 远小于 0.965 → 被排除 ✓

Starobinsky (R+R²): n_s = 1 - 2/N, r = 6/N²
当 N = 60: n_s ≈ 0.967, r ≈ 0.0017 → 与观测一致 ✓
```

## 6. 整个计算链的总结

```
1. 势能 V(φ) 给定
2. 计算 ε_V, η_V (代数)
3. 计算 N = ∫(V/V')dφ/M_pl² (积分)
4. 反解 φ(N) (代数)
5. 代入得 n_s(N), r(N) (代数)
6. 用 N ≈ 50-60 (对应今天可观测尺度 horizon exit 时的 e-folding)
7. 比较 n_s 和 r 与观测
```

**每一步都是代数或简单积分**。考试时公式会给出。关键是要:
1. 知道公式是什么
2. 知道每个参数代表什么
3. 会代入计算
