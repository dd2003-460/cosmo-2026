### 模块一：FLRW度规与弗里德曼方程 ⭐⭐⭐⭐⭐（绝对地基）

📖 **详细讲解**: [详细讲解-模块1-FLRW与弗里德曼方程.md](详细讲解-模块1-FLRW与弗里德曼方程.md)

**老师说**："弗里德曼方程是全部，一切从这里开始"

#### 1.1 FLRW度规

$$ds^2 = -dt^2 + a^2(t) \left[ \frac{dr^2}{1-Kr^2} + r^2 d\Omega^2 \right]$$

其中 $d\Omega^2 = d\theta^2 + \sin^2\theta\, d\varphi^2$。

- 度规来源：最大对称空间（6个Killing矢量：3个平移+3个旋转）→ 常曲率三维空间 → 线元必可写成上述形式（Misner-Thorne-Wheeler §22.6）。
- $K=+1,0,-1$ 分别对应封闭、平坦、开放宇宙。
- $a(t)$ 为尺度因子，描述空间整体膨胀。

#### 1.2 弗里德曼方程

$$\left(\frac{\dot{a}}{a}\right)^2 = H^2 = \frac{8\pi G}{3}\rho - \frac{K}{a^2} + \frac{\Lambda}{3}$$

$$\frac{\ddot{a}}{a} = -\frac{4\pi G}{3}(\rho+3p) + \frac{\Lambda}{3}$$

**来源与动机**: 将 FLRW 度规代入 Einstein 场方程 $G_{\mu\nu}+\Lambda g_{\mu\nu}=8\pi G\,T_{\mu\nu}$，计算 Christoffel 符号 → Riemann 张量 → Ricci 张量 → Einstein 张量，取 $00$ 分量和 $ii$ 分量即得。物理直觉上：
- 第一个方程是"能量约束"：Hubble 率由物质密度和曲率决定。
- 第二个方程决定加速度：$\rho+3p$ 是主动引力密度，压强也产生引力效应（相对论性物质 $p=\rho/3$，引力比牛顿更强）。

#### 1.3 能量守恒方程

$$\dot{\rho}+3H(\rho+p)=0$$

**来源**: 从 $\nabla_\mu T^{\mu\nu}=0$ 在 FLRW 度规下算出。例如取 $\nu=0$ 分量，$\nabla_\mu T^{\mu 0} = \partial_t\rho + 3H(\rho+p) = 0$。

**物理含义**: 宇宙膨胀时体积增大，密度下降。辐射因波长也被拉长，额外损失能量，故 $\rho_r\propto a^{-4}$；物质仅因体积稀释，$\rho_m\propto a^{-3}$；暗能量 $\rho_\Lambda=\text{const}$。

#### 1.4 临界密度与密度参数

$$\rho_c = \frac{3H^2}{8\pi G}, \quad \Omega \equiv \frac{\rho}{\rho_c}, \quad \Omega_K = 1-\Omega$$

**来源**: 将弗里德曼方程改写为 $\Omega-1 = \frac{K}{a^2H^2}$，定义 $\Omega_c$ 使右端为零时的密度。这是标准教材（Peacock §7; Dodelson §2）中的通用定义。

#### 1.5 状态方程

- 辐射: $p=\rho/3$ → $\rho\propto a^{-4}$
- 物质（非相对论）: $p=0$ → $\rho\propto a^{-3}$
- 暗能量（宇宙学常数）: $p=-\rho$ → $\rho=\text{const}$

### 模块二：慢滚暴胀与标量场方程 ⭐⭐⭐⭐⭐

📖 **详细讲解**: [详细讲解-模块2-慢滚暴胀与标量场方程.md](详细讲解-模块2-慢滚暴胀与标量场方程.md)

**老师说**："重点是功率谱公式及计算"

#### 2.1 标量场的作用量与运动方程

$$S = \int d^4x\sqrt{-g}\left[-\frac{1}{2}g^{\mu\nu}\partial_\mu\phi\,\partial_\nu\phi - V(\phi)\right]$$

在 FLRW 度规下变分得到运动方程:

$$\ddot{\phi}+3H\dot{\phi}+V'(\phi)=0$$

**来源与动机**: $\Box\phi-V'(\phi)=0$，其中 $\Box\phi = \frac{1}{\sqrt{-g}}\partial_\mu(\sqrt{-g}g^{\mu\nu}\partial_\nu\phi)$。在 FLRW 度规下 $\sqrt{-g}=a^3$，空间梯度为零（均匀场），得到 $\ddot{\phi}+3H\dot{\phi}+V'(\phi)=0$。

**"Hubble 摩擦"**: 项 $3H\dot{\phi}$ 类比过阻尼振子的摩擦项 $2\gamma\dot{\phi}$，当 $3H\gg\ddot{\phi}$ 时，$\phi$ 缓慢滚动（slow-roll）。

#### 2.2 慢滚参数

$$\epsilon_V = \frac{M_{Pl}^2}{2}\left(\frac{V'}{V}\right)^2, \quad \eta_V = M_{Pl}^2\frac{V''}{V}$$

其中 $M_{Pl}=(8\pi G)^{-1/2}\approx 2.435\times 10^{18}\text{ GeV}$ 为约化普朗克质量。

慢滚条件: $\epsilon_V\ll 1$ 且 $|\eta_V|\ll 1$。

**动机**: 无量纲化确保参数与具体势无关，是判断暴胀持续时长的自然度量。

#### 2.3 暴胀三疑难

**① 视界问题**: 共动 horizon $d_H(t) = \int_0^t \frac{c\,dt'}{a(t')}$。暴胀期间 $a(t)$ 指数增长导致 $\int dt'/a(t')$ 收敛，早期共动 horizon 远小于现在。

**② 平坦性问题**: 由 $|1-\Omega| \propto a$（物质主导）或 $|1-\Omega|\propto a^2$（辐射主导），远离观测值 $\Omega\approx 1$。暴胀使 $\Omega\to 1$。

**③ 磁单极问题**: 暴胀将早期高能相变产生的 relic 稀释到可忽略。

#### 2.4 典型计算：$V(\phi)=\frac{1}{2}m^2\phi^2$

- $\epsilon_V = \frac{2M_{Pl}^2}{\phi^2}$, $\eta_V = \frac{M_{Pl}^2}{\phi^2}$
- 慢滚结束于 $\phi_{\text{end}} = \sqrt{2}\,M_{Pl}$
- e-folds 数: $N = \int H\,dt = \int \frac{V}{M_{Pl}^2V'} d\phi = \frac{\phi^2-\phi_{\text{end}}^2}{4M_{Pl}^2}$
- 当 $N=60$ 时，$\phi \approx \sqrt{4N+2}\,M_{Pl}\approx 15.6\,M_{Pl}$

#### 2.5 谱指数与张标比

$$n_s-1 = 2\eta_V-6\epsilon_V, \quad n_T = -2\epsilon_V, \quad r = \frac{\mathcal{P}_T}{\mathcal{P}_\mathcal{R}} = 16\epsilon_V$$

**观测值**（Planck 2018 + BK15）: $n_s = 0.9649\pm 0.0042$ (68% CL), $r_{0.002}<0.056$ (95% CL).

### 模块三：原初扰动功率谱 ⭐⭐⭐⭐⭐（考试核心！）

📖 **详细讲解**: [详细讲解-模块3-原初扰动功率谱.md](详细讲解-模块3-原初扰动功率谱.md)

#### 3.1 标量功率谱

$$\mathcal{P}_\mathcal{R}(k) = \frac{1}{8\pi M_{Pl}^2\epsilon_H}\left(\frac{H}{2\pi}\right)^2\bigg|_{k=aH}$$

**来源**: 从 Mukhanov-Sasaki 变量 $v=z\mathcal{R}$ 出发，求解弯曲时空量子场方程，取 Bunch-Davies 真空初态，在 horizon crossing ($k=aH$) 处冻结振幅 $|\mathcal{R}_k|\approx H/(2\sqrt{2}\epsilon_H M_{Pl})$，平方后乘 $k^3/(2\pi^2)$ 即得功率谱。

**动机**: 扰动在 horizon 内是振荡的量子涨落；在 horizon 外冻结为经典扰动。$\mathcal{R}$ 是规范不变量（Maldacena 守恒定理保证了超 horizon 后的守恒性）。

#### 3.2 张量功率谱

$$\mathcal{P}_T(k) = \frac{2}{\pi}\frac{H^2}{M_{Pl}^2}\bigg|_{k=aH}$$

#### 3.3 谱指数详细计算

从 $\mathcal{P}_\mathcal{R}\propto H^2/(\epsilon_H M_{Pl}^2)$ 出发：

$$n_s-1 = \frac{d\ln\mathcal{P}_\mathcal{R}}{d\ln k} = 2\frac{d\ln H}{d\ln k}-\frac{d\ln\epsilon_H}{d\ln k}$$

利用 $\frac{d}{d\ln k} = \frac{\dot{\phi}}{\epsilon_H M_{Pl}^2 H}\frac{d}{d\phi}$（slow-roll 下），和 $\epsilon_H\simeq\epsilon_V$，逐次展开得到：

$$n_s-1 \simeq 2\eta_V-6\epsilon_V$$

#### 3.4 幂律势 $V=V_0\phi^p$ 的解析结果

$$\epsilon_V = \frac{p^2M_{Pl}^2}{2\phi^2}, \quad \eta_V = \frac{p(p-1)M_{Pl}^2}{\phi^2}$$

以 $p=2$ 为例 ($V=\frac{1}{2}m^2\phi^2$)：
- $n_s-1 = \frac{2-8N}{2N+1}$ → $N=60$ 时 $n_s\approx 0.967$
- $r = \frac{8}{N} \to N=60$ 时 $r\approx 0.133$

### 模块四：再加热与原始核合成 ⭐⭐⭐

📖 **详细讲解**: [详细讲解-模块4-再加热与原始核合成.md](详细讲解-模块4-再加热与原始核合成.md)

#### 4.1 再加热

暴胀场 $\phi$ 在势能谷底振荡 → 通过 Yukawa/规范耦合衰变到标准模型粒子 → 热化。

**再加热温度**:

$$T_{rh} \simeq \left(\frac{90}{\pi^2 g_*}\right)^{1/4}\sqrt{\Gamma_\phi M_{Pl}}$$

**热平衡条件**: $\Gamma_{\text{react}} > H$。当反应率低于 Hubble 率时，粒子"冻结"（freeze-out）。

#### 4.2 原始核合成（BBN）

时间 $t\sim 1$ 秒，温度 $T\sim 1$ MeV。

**关键步骤**:
1. $n\leftrightarrow p$ 通过弱相互作用维持平衡: $n \leftrightarrow p+e^-+\bar{\nu}_e$
2. $T\sim 0.8$ MeV 时 freeze-out，$n/p\simeq 1/6$
3. 氘瓶颈: $D$ 形成于 $T\sim 0.1$ MeV（光致分解阈值）
4. 无 $A=5,8$ 稳定核 → 瓶颈 → 主要产物是 $^4$He

**丰度**: $Y_p\approx 0.25$ (He-4 质量分数)，D/H$\sim 10^{-5}$，Li-7 有待商榷（锂问题）。

### 模块五：宇宙学距离、CMB与大尺度结构 ⭐⭐⭐⭐

📖 **详细讲解**: [详细讲解-模块5-宇宙学距离与CMB大尺度结构.md](详细讲解-模块5-宇宙学距离与CMB大尺度结构.md)

#### 5.1 距离定义

**共动距离**（来自光子的 null geodesic $ds^2=0$）:

$$\chi(z) = \int_0^z\frac{c\,dz'}{H(z')}$$

**角直径距离**:

$$D_A(z) = \frac{1}{1+z}\cdot\frac{1}{H_0\sqrt{|\Omega_K|}}\cdot S_K\!\left(\sqrt{|\Omega_K|}\,\chi(z)\right)$$

其中 $S_K(x)=\sin x$ ($K>0$), $x$ ($K=0$), $\sinh x$ ($K<0$)。

**光度距离**:

$$D_L(z) = (1+z)^2 D_A(z) = (1+z)\chi(z) \quad (\text{平坦宇宙 } K=0)$$

**距离模数**: $\mu = 5\log_{10}(D_L/\text{Mpc}) + 25$

#### 5.2 CMB 角功率谱

温度涨落展开:

$$\frac{\Delta T}{T}(\hat{n}) = \sum_{\ell m} a_{\ell m}Y_{\ell m}(\hat{n}), \quad C_\ell = \langle |a_{\ell m}|^2 \rangle$$

第一峰位置 $\ell\sim 200$ → 对应声波 horizon 角尺度 → 观测表明 $\Omega_K\approx 0$（平坦宇宙）。

#### 5.3 物质功率谱

$$P(k) = A_s\,k^{n_s}\,T^2(k)$$

$T(k)$ 为转移函数，包含声学振荡和 Silk 阻尼的特征。

#### 5.4 等时红移

$$z_{eq} \simeq 3400$$

（由 $\Omega_m h^2 \simeq \Omega_r h^2$ 在 $T_{CMB}=2.725$ K 计算得到）。

## 📖 模块间衔接总览

|----|----|---------|
