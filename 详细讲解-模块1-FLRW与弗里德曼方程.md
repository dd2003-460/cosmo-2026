# 宇宙学详细讲解 — 模块1: FLRW度规与弗里德曼方程

## 模块定位

这是整个现代宇宙学的**地基**。没有FLRW度规，就没有弗里德曼方程；没有弗里德曼方程，就无法描述宇宙演化；没有宇宙演化，就无法讨论暴涨、扰动、CMB。

**整个逻辑链**:
```
哥白尼原理(均匀+各向同性)
    → FLRW度规(时空几何)
    → 爱因斯坦场方程
    → 弗里德曼方程(宇宙动力学)
    → 尺度因子a(t)的时间演化
    → 红移、距离、年龄等观测预言
```

---

## 1. 从对称性到度规: FLRW度规是怎么想到的？

### 物理直觉

**核心问题**: 宇宙在大尺度上看起来 everywhere 差不多（没有中心，没有特殊方向）。如何用数学描述？

**思考路径**:

**Step 1**: 把时空分成"时间"和"空间"
```
d s² = g_μν dx^μ dx^ν

我们需要一个分割: 时间坐标 t + 空间坐标 x^i
为什么可以这样分? 因为宇宙整体均匀 — 每个观测者看到的情况一样，
所以存在一个"共动观测者"的概念，他们感受不到任何运动。
```

**动机**: 如果宇宙各向同性，那么存在一类观测者，他们对所有人来说看到的天空都一样 — 这就是共动观测者。他们的固有时就是宇宙时 t。

**Step 2**: 空间部分的最大对称性
```
三维空间如果是最大对称的（均匀+各向同性），那么它的曲率必须是常数。

回忆微分几何: 常曲率三维空间的度规只能是:
ds_3² = a²(t) × [dr²/(1-Kr²) + r²(dθ² + sin²θ dφ²)]

其中:
- K = +1: 正曲率（3-球面 S³）
- K = 0: 零曲率（平直 R³）
- K = -1: 负曲率（双曲空间 H³）
- a(t): 尺度因子，可以随时间变化（但空间各点相同 → 均匀性）
```

**动机**: 常曲率空间的分类是标准微分几何结果。物理学家直接套用。

**Step 3**: 时间-空间交叉项为零
```
如果存在空间各向同性，那么不能有 t-r 交叉项（否则某个方向特殊）。
所以 g_0i = 0。
```

**动机**: 各向同性禁止任何方向性 — 包括时间方向和径向之间的关联。

**Step 4**: 时间部分
```
g_00 = -1（因为我们用共动观测者的固有时，即宇宙时）
```

**结果 — FLRW 度规**:
```
ds² = -dt² + a²(t) [dr²/(1-Kr²) + r²(dθ² + sin²θ dφ²)]    (FLRW)
```

### 为什么这样想？

- **不是瞎猜出来的** — 是从对称性原理严格推导出来的
- 类比: 电磁学中从 U(1) 规范对称性 → 唯一可能的拉氏量是 F_μνF^μν
- FLRW 是从"宇宙学原理"这个对称性假设 → 唯一可能的时空几何

### 物理图像

想象一个橡皮气球，表面画满点。气球膨胀 → 点之间距离增加。但每个点看到的邻居分布是一样的（均匀）。这就是 K=0 或 K=+1 的情况。a(t) 就是气球的半径。

## 2. 从爱因斯坦方程到弗里德曼方程

### 物理直觉

**核心问题**: 知道了几何（FLRW度规），怎么知道物质怎么演化？

**思考路径**:

**Step 1**: 爱因斯坦场方程
```
G_μν + Λg_μν = 8πG T_μν

左边 = 几何（爱因斯坦张量）
右边 = 物质（应力-能量张量）
```

**动机**: 广义相对论的基本方程。几何 = 物质。这是已知的物理定律。

**Step 2**: 建模物质
```
宇宙中的物质近似为完美流体:
T_μν = (ρ + p)u_μ u_ν + p g_μν

其中:
- ρ(t) = 能量密度（只依赖时间 → 均匀性）
- p(t) = 压强
- u^μ = (1, 0, 0, 0)（共动观测者，静止在坐标中）
```

**动机**: 大尺度上宇宙看起来像流体。没有粘滞、没有热传导 → 完美流体近似。

**Step 3**: 计算 Ricci 张量和爱因斯坦张量
```
这一步是纯计算。代入 FLRW 度规，计算 Christoffel 符号 → Riemann 张量 → Ricci 张量 → Einstein 张量。

结果（关键步骤）:

G_00 = 3[(ȧ/a)² + K/a²] = 3[H² + K/a²]

G_ii = -(2ä/a + ȧ²/a² + K/a²) g_ii

这里:
- H = ȧ/a = Hubble 参数
- ȧ = da/dt, ä = d²a/dt²
- 下标 i 表示空间分量
```

**动机**: 这不是"想到"的，是"算出来"的。学生需要知道的是**为什么代进去就能得到结果**，而不是记住计算细节。考试时公式会给出，所以关键是理解物理含义。

**Step 4**: 写出 00 和 ii 分量方程

**00 分量**（第一个弗里德曼方程）:
```
G_00 + Λg_00 = 8πG T_00
3[H² + K/a²] + Λ = 8πG ρ

整理:
H² = (8πG/3)ρ - K/a² + Λ/3    (弗里德曼方程 #1)

物理含义: 膨胀速率由能量密度和曲率决定。
- ρ 大 → H 大（物质引力促使更快膨胀，因为早期宇宙中物质多时膨胀也快）
- K > 0 → H 小（正曲率"减速"膨胀，几何上球面"卷回去"）
- Λ > 0 → H 大（暗能量驱动加速膨胀）
```

**ii 分量**（第二个弗里德曼方程）:
```
G_ii + Λg_ii = 8πG T_ii

-(2ä/a + ȧ²/a² + K/a²) + Λ = -8πG p

利用方程1消去 ȧ²/a² + K/a²:

2ä/a = - (8πG/3)ρ + Λ/3 + 8πGp

整理:
ä/a = -(4πG/3)(ρ + 3p) + Λ/3    (弗里德曼方程 #2)

物理含义: 加速度由 (ρ + 3p) 决定！
- ρ > 0, p ≥ 0 → ä < 0（引力是吸引的，减速膨胀）
- 但 p < -ρ/3 时（如暗能量或暴胀子势能）→ ä > 0（加速膨胀）
```

### 为什么 (ρ + 3p) 出现？

这是广义相对论和牛顿引力的关键区别。在 GR 中，**压强也产生引力**。

- 辐射: p = ρ/3 → (ρ + 3p) = 2ρ → 引力效果是物质的2倍
- 物质: p ≈ 0 → (ρ + 3p) ≈ ρ
- 暗能量: p ≈ -ρ → (ρ + 3p) ≈ -2ρ → 反引力（加速）

### 流体力学方程（能量守恒）

从 ∇_μ T^μν = 0 自动得到:
```
ρ̇ + 3H(ρ + p) = 0    (能量守恒)

物理含义:
- 3Hρ: 体积膨胀导致密度稀释（类似绝热膨胀）
- 3Hp: 压强做功导致的能量变化
```

**记忆方法**: "3H" 来自三维空间的膨胀速率 Tr(∇v) = 3H。

## 3. 各成分密度的演化

### 从能量守恒推导

```
已知: ρ̇ + 3H(ρ + p) = 0

对于每种成分，有状态方程: p = wρ

(1) 物质: w = 0 → ρ̇ + 3Hρ = 0
    → dρ/ρ = -3da/a (因为 H = ȧ/a, Hdt = da/a)
    → lnρ = -3lna + const
    → ρ_m ∝ a⁻³ ✓

(2) 辐射: w = 1/3 → ρ̇ + 4Hρ = 0
    → ρ_r ∝ a⁻⁴ ✓

(3) 宇宙学常数: w = -1 → ρ̇ = 0
    → ρ_Λ = const ✓
```

### 为什么这样依赖？

- **物质 ρ ∝ a⁻³**: 体积增大 → 密度下降。简单。
- **辐射 ρ ∝ a⁻⁴**: 除了体积稀释(a⁻³)，还有**红移**效应 — 每个光子的能量 E = hν ∝ 1/a。所以额外的 a⁻¹。
- **暗能量 ρ = const**: 它是真空能，不随体积稀释。体积增大，总能量增大 → 密度不变。

### 临界密度

```
定义: ρ_c = 3H²/(8πG)

为什么这样定义? 因为如果 K = 0（平坦宇宙）:
H² = (8πG/3)ρ → ρ = 3H²/(8πG) = ρ_c

所以 ρ_c 是平坦宇宙所需的密度。

定义密度参数: Ω = ρ/ρ_c

则弗里德曼方程变为:
1 = Ω + Ω_K, 其中 Ω_K = -K/(a²H²)

观测: Ω_total ≈ 1 (平坦宇宙)
→ Ω_m + Ω_Λ ≈ 1
→ Ω_m ≈ 0.3, Ω_Λ ≈ 0.7
```

## 4. 这道题怎么考？

老师明确说:"公式会给出，考代入计算"。

**典型考法**:

1. "给定 ρ = ρ_m + ρ_r，求 H(z)"
   → 用 ρ_m = ρ_m0(1+z)³, ρ_r = ρ_r0(1+z)⁴
   → H²(z) = H₀²[Ω_r0(1+z)⁴ + Ω_m0(1+z)³ + Ω_Λ0]

2. "物质-辐射相等时的红移"
   → ρ_m(a_eq) = ρ_r(a_eq)
   → ρ_m0 a_eq⁻³ = ρ_r0 a_eq⁻⁴
   → a_eq = ρ_r0/ρ_m0 = Ω_r0/Ω_m0
   → 1 + z_eq = Ω_m0/Ω_r0 ≈ 3400

3. "给定 Ω_m, Ω_Λ，求宇宙年龄"
   → t = ∫₀^∞ dz/[(1+z)H(z)]
