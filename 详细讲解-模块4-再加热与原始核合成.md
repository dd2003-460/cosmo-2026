# 宇宙学详细讲解 — 模块4: 再加热与原始核合成

## 模块定位

暴胀完成后，暴胀场的巨大能量必须以某种方式转化为普通粒子，才能开启热大爆炸时代。这个过程叫做**再加热**（reheating）。

暴胀能量转化之后，宇宙经历了几个关键阶段：
1. **辐射为主**：粒子产生并热化
2. **原始核合成（BBN）**：轻元素形成
3. **物质-辐射等时**：结构开始形成

**逻辑链**:
```
暴胀场φ缓慢滚下势能
    → 暴胀结束，φ开始振荡
    → φ衰减为普通粒子（再加热）
    → 宇宙充满热辐射
    → 核合成（BBN）
    → 轻元素丰度（D, He, Li）
    → 与观测比较验证大爆炸模型
```

## 1. 暴胀结束与场的振荡

### 1.1 慢滚失效

慢滚条件 $\epsilon_V \ll 1$ 和 $|\eta_V| \ll 1$ 逐渐被破坏。当 $\phi$ 滚到势能更陡的区域时，$\epsilon_V \sim 1$，慢滚近似失效。

**慢滚结束的判断**:

$$\epsilon_V(\phi_{\text{end}}) = 1 \quad \text{或} \quad |\eta_V(\phi_{\text{end}})| = 1 \tag{1}$$

此时 $\ddot{\phi}$ 不再能忽略，标量场开始加速滚下势能。

### 1.2 振荡行为

当 $\phi$ 到达势能最小值附近时，它会围绕最小值振荡。

以 $V(\phi) = \frac{1}{2}m^2\phi^2$ 为例，最小值在 $\phi = 0$。振荡时:

$$\phi(t) \approx \Phi(t) \cos(mt)$$

其中振幅 $\Phi(t)$ 随时间衰减。

**平均状态方程**:

对于 $V \propto \phi^p$:
- $p = 2$（二次势）: 平均 $p = 0$ → 表现像**物质**
- $p = 4$（四次势）: 平均 $p \approx \rho/3$ → 表现像**辐射**

**动机**: 这是 Virial 定理的直接结果。势能在振荡中对时间的平均等于动能的平均。对于 $V \propto \phi^p$，动能和势能的平均满足 $\langle p_\phi \rangle = \frac{p-2}{p+2} \langle \rho_\phi \rangle$。

当 $p=2$: $\langle p_\phi \rangle = 0$（物质）
当 $p=4$: $\langle p_\phi \rangle = \rho_\phi/3$（辐射）

### 1.3 膨胀行为

在振荡期间，如果 $p = 2$（表现像物质）:

$$a(t) \propto t^{2/3}, \quad \rho_\phi \propto a^{-3}$$

这与物质主导时期的演化相同。**动机**: 振荡标量场的平均状态方程 $w = 0$ 正好对应物质。

## 2. 再加热过程

### 2.1 什么是再加热？

暴胀场的能量必须转化为标准模型粒子，才能产生热辐射背景。这个过程就是再加热。

**再加热温度** $T_{rh}$:

再加热完成时宇宙达到的温度。它是宇宙学中的一个重要参数，影响了暴胀与可观测宇宙之间的联系。

### 2.2 衰变机制

**简单衰变模型**:

暴胀场通过衰变产生粒子:

$$\phi \to \text{标准模型粒子}$$

衰变率:

$$\Gamma_\phi \sim m_\phi$$

其中 $m_\phi$ 是暴胀场的质量（势能的二阶导数在最小值处的平方根）:

$$m_\phi = \sqrt{V''(\phi_{\text{min}})} \tag{2}$$

**物理动机**: 在粒子物理中，耦合常数为 $g$ 的衰变率一般形式为 $\Gamma \sim g^2 m_\phi / (8\pi)$。对于重力强度的耦合 $g \sim M_{Pl}^{-1}$，有 $\Gamma_\phi \sim m_\phi^3 / M_{Pl}^2$。

### 2.3 参数共振（参量共振）

90年代的重要发展：再加热可能不是简单的衰变，而是**参数共振**过程。

**物理机制**:

当 $\phi$ 在势阱中振荡时，它与其它场的耦合导致其它场的有效质量随时间振荡。当振荡频率满足特定条件时，粒子数指数增长。

**Mathieu 方程**:

在耦合场 $\chi$ 的情况下，运动方程化为:

$$\chi_k'' + \left[ k^2 - A + 2q \cos(2z) \right] \chi_k = 0$$

这是**Mathieu 方程**。当参数 $A$ 和 $q$ 处于不稳定带时，$\chi_k$ 指数增长:

$$\chi_k \propto e^{\mu_k z}$$

其中 $\mu_k$ 是增长指数。

**物理图像**: 想象一个摆，其悬挂点以特定频率振动。当振动频率与摆的自然频率有特定关系时，摆的振幅指数增长。这就是参数共振。

**动机**: 参数共振比简单衰变效率高得多。在共振期间，暴胀场的能量可以在几个振荡周期内就转化为其它粒子。

### 2.4 再加热温度的估计

再加热完成的条件:

$$\Gamma_\phi \sim H(T_{rh}) \tag{3}$$

其中 $H(T_{rh})$ 是温度 $T_{rh}$ 时的 Hubble 参数。

从弗里德曼方程:

$$H = \sqrt{\frac{8\pi G}{3} \rho_r} = \sqrt{\frac{8\pi^3 g_*}{90}} \frac{T_{rh}^2}{M_{Pl}}$$

其中 $g_*$ 是有效自由度。

所以:

$$T_{rh} \sim \sqrt{\Gamma_\phi M_{Pl}} \tag{4}$$

**典型值**:

- 如果 $\Gamma_\phi \sim 10^{-6} M_{Pl}$: $T_{rh} \sim 10^9$ GeV
- 如果 $\Gamma_\phi \sim m_\phi \sim 10^{13}$ GeV: $T_{rh} \sim 10^{15}$ GeV

**动机**: 再加热温度对宇宙学后果有重要影响。它决定了:
1. 大统一理论（GUT）相变后的单极是否被充分稀释
2. 暗物质粒子是否能被充分产生
3. 原初引力波谱的形状

### 2.5 暴胀与可观测宇宙的联系

暴胀结束时 horizon exit 的模到今天对应的 e-folding 数 $N$ 为:

$$N \approx 55 - 65 \tag{5}$$

具体数值取决于再加热历史。估计公式:

$$N \approx 57.6 + \frac{1}{4}\ln\left(\frac{V_{\text{end}}}{V_{\text{exit}}}\right) + \frac{1-3w_{rh}}{4(1+w_{rh})}\ln\left(\frac{\rho_{rh}}{\rho_{\text{end}}}\right)$$

其中 $w_{rh}$ 是再加热期间的状态方程参数。

**动机**: 这给出了暴胀能标与可观测参数之间的定量联系。暴胀期间 horizon exit 的模对应 CMB 上的角尺度 $\sim 1^\circ$。

## 3. 热平衡与反应率

### 3.1 反应率与 Hubble 率的竞争

早期宇宙中，粒子之间的反应速率与宇宙的膨胀速率竞争。

**反应率**:

$$\Gamma_{\text{react}} = n \langle \sigma v \rangle \tag{6}$$

其中:
- $n$ 是粒子数密度
- $\langle \sigma v \rangle$ 是反应截面与相对速度的乘积的平均

**Hubble 率**:

$$H = \sqrt{\frac{8\pi G}{3} \rho}$$

**物理竞争**:
- $\Gamma_{\text{react}} > H$: 反应速度快于膨胀 → **热平衡**
- $\Gamma_{\text{react}} < H$: 反应速度慢于膨胀 → **冻结**（freeze-out 或 freeze-in）

**动机**: 这是早期宇宙动力学的核心概念。粒子的丰度由 $\Gamma/H$ 决定。

### 3.2 冻结过程

**冻结-out（WIMP 暗物质）**:

如果 $\Gamma_{\text{react}}$ 随温度下降而降低（因为 $n$ 降低），当 $\Gamma < H$ 时，粒子丰度"冻结"在一个常数/温度。

**冻结-in（轴子、引力子）**:

如果反应率始终很低，粒子通过缓慢的生成过程逐渐积累。

**动机**: 冻结过程决定了宇宙中各种粒子的剩余丰度。这是暗物质、中微子、轻元素丰度计算的核心。

## 4. 原始核合成（Big Bang Nucleosynthesis, BBN）

### 4.1 BBN 的时间与温度

**时间**: $t \sim 1$ 秒
**温度**: $T \sim 1$ MeV（约 $10^{10}$ K）

在这个时刻，宇宙足够冷，中子和质子可以结合成氘核。在此之前，高能光子会立即分解任何形成的氘核（光致分解）。

### 4.2 $n \leftrightarrow p$ 转换反应

$$n + \nu_e \leftrightarrow p + e^-$$

$$n + e^+ \leftrightarrow p + \bar{\nu}_e$$

$$n \leftrightarrow p + e^- + \bar{\nu}_e \quad (\text{自由衰变})$$

在 $T \gg m_e$ 时，这些反应保持中子-质子平衡:

$$\frac{n_n}{n_p} = e^{-\Delta m / T} \tag{7}$$

其中 $\Delta m = m_n - m_p \approx 1.29$ MeV 是中子-质子质量差。

**动机**: 在热平衡中，较重的粒子（中子）被指数抑制。这是 Boltzmann 分布的直接结果。

### 4.3 冻结

当 $T \sim 0.8$ MeV 时，$n \leftrightarrow p$ 反应率 $\Gamma < H$，中子-质子比"冻结":

$$\left(\frac{n_n}{n_p}\right)_{\text{freeze}} \approx e^{-\Delta m / T_{\text{freeze}}} \approx 0.15 - 0.2$$

随后自由中子继续衰变（半衰期 $\sim 880$ 秒），使中子丰度进一步降低:

$$X_n(t_{\text{BBN}}) \approx 0.13 - 0.14$$

### 4.4 氘瓶颈（Deuterium Bottleneck）

**为什么 BBN 不能立刻合成重元素？**

因为氘核的结合能较低（$B_D = 2.23$ MeV），而宇宙早期的高温光子可以轻易分解氘核:

$$\gamma + D \to p + n$$

在数密度高的光子背景下，即使光子的平均能量远小于氘的结合能，高能尾巴的光子仍然足够多，可以维持光致分解。

**氘瓶颈直到** $T \sim 0.1$ MeV 时才打开。此时高能光子的数密度足够低，氘核可以稳定存在。

### 4.5 从氘到氦-4

一旦氘稳定，核反应链迅速进行:

$$D + p \to {}^3\text{He} + \gamma$$

$$D + n \to {}^3\text{H} + \gamma$$

$$^3\text{He} + n \to {}^4\text{He} + \gamma$$

$$^3\text{H} + p \to {}^4\text{He} + \gamma$$

几乎所有剩余的中子都被纳入氦-4核中（因为氦-4是最稳定的轻核）。

**氦-4丰度**:

$$Y_p \equiv \frac{4n_{He}}{n_B} \approx 2 \frac{n_n}{n_B} \approx 0.25 \tag{8}$$

其中 $n_B$ 是重子数密度。

**动机**: $Y_p \approx 0.25$ 是 BBN 最著名的预言，与观测高度一致。这是大爆炸模型最强有力的验证之一。

### 4.6 其他轻元素

- **氘（D）**: $D/H \sim 10^{-5}$ — 对重子密度非常敏感
- **氦-3（$^3$He）**: 丰度低
- **锂-7（$^7$Li）**: $^7\text{Li}/\text{H} \sim 10^{-10}$ — 存在"锂问题"（观测值比理论低约 3 倍）

### 4.7 质量数 $A = 5, 8$ 瓶颈

**为什么 BBN 只能产生轻元素？**

因为不存在稳定的 $A = 5$ 或 $A = 8$ 的核:
- $^5$He, $^5$Li: 不稳定（半衰期 $\sim 10^{-21}$ s）
- $^8$Be: 不稳定（半衰期 $\sim 10^{-16}$ s）

**瓶颈效应**:

没有稳定的 $A = 5$ 核，意味着:
$$^4\text{He} + n/p \to ^5\text{He}/^5\text{Li} \text{（不稳定）}$$

无法继续融合形成更重的元素。

同样的，没有稳定的 $A = 8$ 核:
$$^4\text{He} + ^4\text{He} \to ^8\text{Be} \text{（不稳定）}$$

**三体过程 $^4\text{He} + ^4\text{He} + ^4\text{He} \to ^{12}\text{C}$（三重 α 过程）**:

在 BBN 的条件下，三体过程的概率极低，无法有效进行。这需要恒星内部的高温高压环境。

**动机**: 这就是为什么 BBN 产生的重元素只到锂为止。恒星内部才是合成重元素的场所。

## 5. BBN 的观测验证

### 5.1 氦-4丰度

**观测**: $Y_p^{\text{obs}} = 0.245 \pm 0.003$

**理论**: $Y_p \approx 0.247$（取决于重子密度）

**一致性**: 极好 ✓

### 5.2 氘丰度

**观测**: $(D/H)^{\text{obs}} \approx 2.5 \times 10^{-5}$

**理论**: $(D/H)^{\text{th}} \propto \Omega_b^{-1.6}$（对重子密度高度敏感）

**一致性**: 极好 ✓ 氘丰度是重子密度的精确探针。

### 5.3 BBN 对宇宙学的意义

1. **验证大爆炸模型**: BBN 预言的轻元素丰度与观测高度一致
2. **测量重子密度**: $\Omega_b h^2 \approx 0.022$
3. **限制额外自由度**: 额外中微子种类 $N_{\text{eff}} < 3.5$
4. **限制粒子物理**: 对稳定粒子（如暗物质）的性质有约束

## 6. 这个模块怎么考？

**典型考法**:

**1. "再加热过程的物理图像"**

→ 暴胀场振荡、衰变为标准模型粒子、能量转移到热辐射。

**2. "BBN 的时间、温度、主要产物"**

→ $t \sim 1$ s, $T \sim 1$ MeV，主要产物 He-4（~25%丰度）、D、Li-7。

**3. "氘瓶颈的物理原因"**

→ 氘结合能低，高温光子分解氘。直到 $T \sim 0.1$ MeV 才打开。

**4. "质量数 A = 5, 8 瓶颈"**

→ 无稳定 A=5 和 A=8 核 → BBN 无法合成更重元素。

**5. "BBN 对宇宙学参数的约束"**

→ 测量 $\Omega_b$，限制 $N_{\text{eff}}$。
