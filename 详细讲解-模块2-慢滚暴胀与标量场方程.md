# 宇宙学详细讲解 — 模块2: 慢滚暴胀与标量场方程

## 模块定位

这是课程的**核心主题**。暴胀解决了热大爆炸的三个疑难，并提供了结构形成的初始扰动。

**整个逻辑链**:
```
热大爆炸疑难(视界/平坦/单极)
    → 需要一个机制让早期宇宙指数膨胀
    → 指数膨胀需要负压源
    → 标量场(暴胀子)具有负压特性 ✓
    → 标量场在FRW背景下的运动方程
    → 慢滚近似 → 可以解析计算
    → 计算原初扰动功率谱
    → 与CMB观测比较 → 确定模型参数
```

## 1. 暴胀解决的三个疑难

### 1.1 视界问题

**问题**: CMB在天空相反两点温度相同（精度 $10^{-5}$）。但这两点在今天的光锥中没有因果联系——光从大爆炸以来还没足够的时间让它们交换信息。

**如何量化这个问题？**

计算**共动 horizon**（粒子 horizon）:

$$d_H(t) = \int_0^t \frac{c\, dt'}{a(t')} = \int_0^\eta d\eta' \tag{1}$$

两点因果联系的充要条件：它们的共动距离 $< d_H(t)$。

在 CMB 最后散射面（$z_* \approx 1100$），共动 horizon 约为:
$$d_H(\eta_*) \approx 0.1 \text{ Mpc (共动)}$$

而天空中相隔 $1^\circ$ 的两点对应的共动距离约为:
$$\chi \approx \chi(\eta_*) \times 1^\circ \approx 100 \text{ Mpc (共动)}$$

$$100 \gg 0.1 \Rightarrow \text{没有因果联系} \Rightarrow \text{但温度相同} \Rightarrow \text{疑难}$$

**暴胀怎么解决？**

暴胀期间 $a(t) \propto e^{Ht}$（近似 de Sitter 膨胀）:

此时共形时间:
$$\eta = \int_0^t \frac{dt'}{a(t')} \approx \int_0^t e^{-Ht'} dt' = \frac{1}{H}(1 - e^{-Ht})$$

当 $t \to \infty$，$\eta$ 收敛到有限值 $1/H$！这意味着**共动 horizon 在暴胀期间变小**。

**物理过程**:
1. 暴胀前：可观测宇宙来自一个很小的因果连通区域
2. 暴胀：这个区域被指数拉伸 → 远大于今天的 horizon
3. 暴胀后：热大爆炸演化 → 到今天

**结论**: 现在看到的整个 CMB 天区，暴胀前是因果连通的。视界问题迎刃而解。✓

**物理直觉**: 没有暴胀时，共动 horizon 随时间增大（$\eta$ 随 $t$ 增大），早期可连通的区域越来越小。有暴胀时，共动 horizon 反而在减小——早期很大的因果连通区域被"甩到 horizon 之外"。

### 1.2 平坦性问题

**问题**: 今天观测到 $\Omega \approx 1$（平坦宇宙）。但从弗里德曼方程:

$$1 - \Omega = \frac{K}{a^2 H^2} \tag{2}$$

在不同主导时期的时间演化:

**物质主导**: $a \propto t^{2/3}$, $H = \frac{2}{3t} \Rightarrow a^2 H^2 \propto t^{-2/3}$
$$|1 - \Omega| \propto t^{2/3} \quad (\text{增长})$$

**辐射主导**: $a \propto t^{1/2}$, $H = \frac{1}{2t} \Rightarrow a^2 H^2 \propto t^{0}$
$$|1 - \Omega| \propto t \quad (\text{增长})$$

这意味着：如果今天 $|1 - \Omega| \approx 0.01$，那么在 Planck 时代 ($t \sim 10^{-43}$ s) $|1 - \Omega| \sim 10^{-60}$。

**为什么初始条件这么精确？** 这就是平坦性问题——初始 $\Omega$ 必须极其接近 1，否则今天的宇宙会远非平坦。

**暴胀怎么解决？**

暴胀：$a \propto e^{Ht} \Rightarrow a^2 H^2$ 指数增长。

$$|1 - \Omega| = \frac{K}{a^2 H^2} \xrightarrow{\text{暴胀}} \text{指数减小} \rightarrow 0$$

不管初始 $\Omega$ 偏离 1 多少，暴胀后 $\Omega$ 极其接近 1。

**类比**: 吹一个大气球，表面看起来越来越平。你的局部观测无法感知曲率。

**定量估算**: 假设暴胀 $N$ 个 e-folds，则 $a$ 增大 $e^N$ 倍:
$$|1 - \Omega|_{\text{end}} \approx |1 - \Omega|_{\text{start}} \times e^{-2N}$$

即使初始偏离很大（如 $|1-\Omega|_{\text{start}} \sim 1$），取 $N \approx 60$:
$$|1 - \Omega|_{\text{end}} \sim e^{-120} \sim 10^{-52}$$

### 1.3 磁单极问题

**问题**: Grand Unified Theory (GUT) 相变应该产生大量磁单极（拓扑缺陷）。Kibble 机制估计在 GUT 相变时，每因果 horizon 体积约产生一个单极。但从未观测到。

**暴胀怎么解决**: 暴胀稀释了单极密度。暴胀前存在的单极被稀释到可观测宇宙中几乎为零。

定量：暴胀后单极数密度 $n_M \propto a^{-3}$ 被压低 $e^{-3N}$ 倍，远小于可观测宇宙中的期望值。

## 2. 标量场在 FRW 背景下的运动方程

### 物理直觉

**核心问题**: 需要一个场，它具有负压，能驱动指数膨胀。为什么选标量场？

**思考路径**:

**Step 1**: 为什么是标量场？

标量场 $\phi(x)$ 是最简单的动力学场。

- **矢量场**有方向性 → 破坏各向同性
- **张量场** → 引力子（已经是几何的一部分）
- **标量场** → 各向同性，不破坏对称性 ✓

而且，标量场的拉氏量形式简单:

$$\mathcal{L} = -\frac{1}{2} g^{\mu\nu} \partial_\mu \phi \partial_\nu \phi - V(\phi) \tag{3}$$

**动机**: 在粒子物理中，标量场已经存在——Higgs 场就是标量场。用它驱动暴胀是自然的推广。标量场在量子场论中是标准的动力学自由度，没有额外的结构性 complication。

**Step 2**: 标量场的应力-能量张量

从拉氏量 (3) 通过 Noether 定理（或直接对度规变分）得到应力-能量张量:

$$T_{\mu\nu} = \partial_\mu \phi \partial_\nu \phi - g_{\mu\nu} \left[ \frac{1}{2} g^{\alpha\beta} \partial_\alpha \phi \partial_\beta \phi + V(\phi) \right] \tag{4}$$

对均匀背景 $\phi = \phi(t)$（$\partial_i \phi = 0$）:

**能量密度**:
$$\rho_\phi = T_{00} = \frac{1}{2}\dot{\phi}^2 + V(\phi) \tag{5}$$

**压强**:
$$p_\phi = \frac{1}{3} T^i_i = \frac{1}{2}\dot{\phi}^2 - V(\phi) \tag{6}$$

**推导细节**: 
$$T_{00} = \dot{\phi}^2 - g_{00}\left[\frac{1}{2}(-\dot{\phi}^2) + V\right] = \dot{\phi}^2 + \frac{1}{2}\dot{\phi}^2 + V = \frac{1}{2}\dot{\phi}^2 + V$$

$$T_{ij} = 0 - g_{ij}V = -g_{ij}V \Rightarrow p = \frac{1}{3}g^{ij}T_{ij} = -V$$

等等——需要加上动能项的贡献。完整计算:
$$T_{ij} = \partial_i \phi \partial_j \phi - g_{ij}\left[\frac{1}{2}g^{\alpha\beta}\partial_\alpha\phi\partial_\beta\phi + V\right] = 0 - g_{ij}\left[-\frac{1}{2}\dot{\phi}^2 + V\right]$$

$$p = \frac{1}{3}g^{ij}T_{ij} = \frac{1}{3} \times 3 \times \left(\frac{1}{2}\dot{\phi}^2 - V\right) = \frac{1}{2}\dot{\phi}^2 - V$$

### 关键发现：势能主导时产生负压

如果 $V(\phi) \gg \frac{1}{2}\dot{\phi}^2$（势能主导）:

$$p_\phi \approx -V(\phi) < 0$$

**状态方程参数**:
$$w \equiv \frac{p}{\rho} \approx \frac{\frac{1}{2}\dot{\phi}^2 - V}{\frac{1}{2}\dot{\phi}^2 + V} \approx -1$$

**为什么 $V \gg \frac{1}{2}\dot{\phi}^2$ 就能加速？**

从弗里德曼方程 (8):
$$\frac{\ddot{a}}{a} = -\frac{4\pi G}{3}(\rho + 3p)$$

$$\rho + 3p = \left(\frac{1}{2}\dot{\phi}^2 + V\right) + 3\left(\frac{1}{2}\dot{\phi}^2 - V\right) = 2\dot{\phi}^2 - 2V$$

如果 $V \gg \frac{1}{2}\dot{\phi}^2$:
$$\rho + 3p \approx -2V < 0 \Rightarrow \ddot{a} > 0$$

**物理直觉**: 势能产生负压，负压产生反引力（加速膨胀）。这和牛顿引力完全不同——在 GR 中，**压强也产生引力**。

### Step 3: 标量场的运动方程

一般 curved spacetime 中标量场的运动方程来自作用量变分:

$$S = \int d^4x \sqrt{-g} \left[ -\frac{1}{2} g^{\mu\nu} \partial_\mu \phi \partial_\nu \phi - V(\phi) \right]$$

欧拉-拉格朗日方程:
$$\frac{1}{\sqrt{-g}} \partial_\mu (\sqrt{-g} g^{\mu\nu} \partial_\nu \phi) - V'(\phi) = 0$$

即:
$$\Box \phi - V'(\phi) = 0 \tag{7}$$

其中 $\Box = \frac{1}{\sqrt{-g}} \partial_\mu (\sqrt{-g} g^{\mu\nu} \partial_\nu)$ 是协变 d'Alembertian 算符。

**展开 $\Box \phi$ 在 FLRW 度规下**:

FLRW 度规: $ds^2 = -dt^2 + a^2(t) d\vec{x}^2$

$$\sqrt{-g} = a^3(t)$$

$$\Box \phi = \frac{1}{a^3} \partial_t (a^3 \partial_t \phi) + \frac{1}{a^2} \nabla^2 \phi$$

$$= \ddot{\phi} + 3\frac{\dot{a}}{a} \dot{\phi} - \frac{1}{a^2} \nabla^2 \phi$$

$$= \ddot{\phi} + 3H\dot{\phi} - \frac{1}{a^2} \nabla^2 \phi$$

所以运动方程:

$$\ddot{\phi} + 3H\dot{\phi} + V'(\phi) - \frac{1}{a^2} \nabla^2 \phi = 0 \tag{8}$$

对于均匀背景 $\phi = \phi(t)$（$\nabla^2 \phi = 0$）:

$$\boxed{\ddot{\phi} + 3H\dot{\phi} + V'(\phi) = 0} \tag{9}$$

这就是**FRW 背景下的标量场运动方程**。

**每一项的物理含义**:
- $\ddot{\phi}$: 场的加速度
- $3H\dot{\phi}$: **"Hubble 摩擦"项**——宇宙膨胀就像在黏性介质中运动
- $V'(\phi)$: 势梯度——驱动力

**类比**: 一个小球在势 $V(\phi)$ 中滚动，同时受到与速度成正比的摩擦。摩擦系数 = $3H$。$H$ 越大，摩擦越大。

**为什么叫"Hubble 摩擦"？**

想象一个球在山上滚动。如果没有摩擦 ($3H = 0$)，球会振荡。如果有大摩擦 ($3H \gg m$，其中 $m$ 是球的质量)，球缓慢滚动（overdamped）。

暴胀的条件就是"大摩擦" → 慢滚近似。

## 3. 慢滚近似

### 从"大摩擦"到近似方程

**思考路径**:

**Step 1**: 慢滚条件

类比中，overdamped 的条件是摩擦项远大于加速度项:

$$3H|\dot{\phi}| \gg |\ddot{\phi}| \tag{10}$$

同时，势能主导:

$$V(\phi) \gg \frac{1}{2}\dot{\phi}^2 \tag{11}$$

这两个条件合起来就是**慢滚条件**。

**Step 2**: 慢滚近似下的运动方程

原方程: $\ddot{\phi} + 3H\dot{\phi} + V' = 0$

慢滚近似: $\ddot{\phi} \ll 3H\dot{\phi}$

$$\Rightarrow 3H\dot{\phi} \approx -V'(\phi) \tag{12}$$

同时，能量密度:

$$\rho \approx V(\phi)$$

从弗里德曼方程:

$$3H^2 \approx \frac{8\pi G}{3} V(\phi) = \frac{V(\phi)}{M_{Pl}^2}$$

$$\Rightarrow H^2 \approx \frac{V(\phi)}{3M_{Pl}^2} \tag{13}$$

其中 $M_{Pl} = (8\pi G)^{-1/2} \approx 2.435 \times 10^{18}$ GeV 是**约化普朗克质量**。

**动机**: 这是约化普朗克质量的标准定义（见 Dodelson, Modern Cosmology, §2.3）。约化普朗克质量出现在宇宙学中是因为爱因斯坦方程中包含 $8\pi G$ 的组合。

**将两个方程联立**，消去 $\dot{\phi}$，得到 $\phi$ 和 $V$ 的直接关系:

$$\dot{\phi} \approx -\frac{V'}{3H} \approx -\frac{V'}{\sqrt{3} \times \sqrt{\frac{8\pi G}{3}} \times \sqrt{V}} = -\frac{V'}{3 M_{Pl} \sqrt{V}}$$

**Step 3**: 定义慢滚参数

为了量化"慢滚"，定义无量纲参数:

$$\epsilon_V \equiv \frac{M_{Pl}^2}{2} \left(\frac{V'}{V}\right)^2 \tag{14}$$

$$\eta_V \equiv M_{Pl}^2 \frac{V''}{V} \tag{15}$$

**为什么这样定义？**

$\epsilon_V$ 衡量势能斜率的平方:
- $V'/V$ 有量纲 $[\text{mass}]^{-1}$
- 乘以 $M_{Pl}^2$ 得到无量纲数
- $\epsilon_V \ll 1$ 等价于 $|V'| \ll V/M_{Pl}$

$\eta_V$ 衡量势能曲率:
- $V''/V$ 有量纲 $[\text{mass}]^2$
- 乘以 $M_{Pl}^2$ 得到无量纲数
- $|\eta_V| \ll 1$ 等价于 $|V''| \ll V/M_{Pl}^2$

**慢滚条件的重新表述**:

条件 (10) 和 (11) 可以重新用 $\epsilon_V$ 和 $|\eta_V|$ 表示:

$$\epsilon_V \ll 1 \quad \text{且} \quad |\eta_V| \ll 1$$

这是更便于计算的表述。

**物理含义**: $\epsilon_V$ 衡量势能的"陡峭程度"，$\eta_V$ 衡量势能的"曲率"。慢滚意味着势能非常平坦——势场变化缓慢，标量场缓慢"滚下"势能。

### Step 4: e-folding 数

暴胀的"强度"用 **e-folding 数** $N$ 衡量:

$$N = \ln\left(\frac{a_{\text{end}}}{a}\right) = \int_t^{t_{\text{end}}} H dt \tag{16}$$

**为什么用 $N$？** 因为暴胀需要至少 $N \approx 60$ e-folds 才能解决疑难（具体数值取决于再加热历史）。

**计算 $N$**:

$$N = \int_{t}^{t_{\text{end}}} H dt = \int_{\phi}^{\phi_{\text{end}}} \frac{H}{\dot{\phi}} d\phi$$

用慢滚近似 (12): $\dot{\phi} \approx -V'/(3H)$:

$$\frac{H}{\dot{\phi}} \approx \frac{H}{-V'/(3H)} = -\frac{3H^2}{V'}$$

用 (13): $3H^2 \approx V/M_{Pl}^2$:

$$\frac{H}{\dot{\phi}} \approx -\frac{V}{M_{Pl}^2 V'}$$

所以:

$$N \approx \int_{\phi_{\text{end}}}^{\phi} \frac{V}{V'} \frac{d\phi}{M_{Pl}^2} \tag{17}$$

**动机**: 把时间积分转换为对场的积分——因为 $\phi$ 是单调变化的（在慢滚下 $\dot{\phi} < 0$），这是一个合法的变量替换。

### 为什么 $\epsilon_V$ 和 $|\eta_V|$ 能控制慢滚？

从运动方程 (12):

$$\dot{\phi} \approx -\frac{V'}{3H}$$

对时间求导:

$$\ddot{\phi} \approx -\frac{V'' \dot{\phi}}{3H} + \frac{V' \dot{H}}{3H^2}$$

用 $\dot{H} = -4\pi G (\rho + p) \approx -4\pi G \frac{1}{2}\dot{\phi}^2 \approx -\frac{V'}{6M_{Pl}^2 H} \times \frac{V'}{3H} = -\frac{V'^2}{18M_{Pl}^2 H^2}$:

$$\ddot{\phi} \approx \frac{V'' V'}{9HM_{Pl}^2} + \frac{V'^3}{54M_{Pl}^4 H^3}$$

慢滚条件 $\ddot{\phi} \ll 3H\dot{\phi}$ 要求:

第一项比 $3H\dot{\phi} = -V'/3$ 小: $\left|\frac{V'' V'}{9HM_{Pl}^2}\right| \ll \left|\frac{V'}{3}\right| \Rightarrow |V''/V| \ll 3H^2 \approx V/(3M_{Pl}^2)$

这给出 $|\eta_V| \ll 1$。

第二项比 $3H\dot{\phi}$ 小，给出 $\epsilon_V \ll 1$。

## 4. 典型势能模型

### 4.1 二次势 $V(\phi) = \frac{1}{2}m^2\phi^2$

$$\frac{V'}{V} = \frac{m^2\phi}{\frac{1}{2}m^2\phi^2} = \frac{2}{\phi}$$

$$\frac{V''}{V} = \frac{m^2}{\frac{1}{2}m^2\phi^2} = \frac{2}{\phi^2}$$

$$\epsilon_V = \frac{M_{Pl}^2}{2} \times \frac{4}{\phi^2} = \frac{2M_{Pl}^2}{\phi^2}$$

$$\eta_V = \frac{2M_{Pl}^2}{\phi^2}$$

慢滚结束于 $\epsilon_V = 1$ 或 $|\eta_V| = 1$:

$$\phi_{\text{end}} = \sqrt{2} M_{Pl}$$

e-folding 数:

$$N \approx \int_{\phi_{\text{end}}}^{\phi} \frac{\phi}{2} \frac{d\phi}{2M_{Pl}^2} = \frac{\phi^2 - \phi_{\text{end}}^2}{4M_{Pl}^2}$$

$$\Rightarrow \phi^2 = \phi_{\text{end}}^2 + 4NM_{Pl}^2 = 2M_{Pl}^2 + 4NM_{Pl}^2 = (4N+2)M_{Pl}^2$$

代入:

$$\epsilon_V = \frac{2}{4N+2} \approx \frac{1}{2N} \quad (\text{当 } N \gg 1)$$

$$\eta_V \approx \frac{1}{2N}$$

### 4.2 四次势 $V(\phi) = \frac{\lambda}{4}\phi^4$

类似计算给出 $\epsilon_V = \frac{8M_{Pl}^2}{\phi^2}$，$\eta_V = \frac{12M_{Pl}^2}{\phi^2}$，需要另行积分。

### 4.3 Starobinsky 模型

这不是标量场模型，而是 $R + R^2$ 引力理论。在 Einstein frame 中等效于一个标量场（标量子，scalaron），其势能形式为:

$$V(\phi) \propto \left(1 - e^{-\sqrt{2/3}\phi/M_{Pl}}\right)^2$$

预言 $n_s \approx 1 - 2/N$, $r \approx 12/N^2$。与观测高度一致。

## 5. 这个模块怎么考？

**典型考法**:

**1. "写出 FRW 背景下标量场的运动方程，解释每一项的物理含义"**

→ $\ddot{\phi} + 3H\dot{\phi} + V'(\phi) = 0$，逐项解释。

**2. "慢滚近似是什么？为什么它能给出加速膨胀？"**

→ $3H\dot{\phi} \approx -V'$, $V \gg \frac{1}{2}\dot{\phi}^2$, $p \approx -V$, $\ddot{a} > 0$。

**3. "给定 $V(\phi)$，计算 $\epsilon_V$ 和 $\eta_V$"**

→ 直接代入定义。

**4. "计算 $N = \int (V/V') d\phi/M_{Pl}^2$"**

→ 对给定势能积分。

**5. "慢滚条件为什么保证足够多的 e-folds？"**

→ 慢滚时间长 → $N$ 大 → 解决视界/平坦问题。

## 6. 下一步：从理想暴胀到实际扰动

模块2给出了暴胀的理想图像：一个均匀标量场在平坦势上缓慢滚动，产生近似 de Sitter 膨胀。

但实际的宇宙不是完全均匀的——我们有星系、星系团。暴胀理论必须解释这些**结构从哪里来**。

答案是：暴胀期间存在**量子涨落**。这些涨落被暴胀拉伸到宏观尺度，成为经典扰动。这就引出了功率谱的计算。
