# 宇宙学详细讲解 — 模块1: FLRW度规与弗里德曼方程

## 模块定位

这是整个现代宇宙学的**地基**。没有FLRW度规，就没有弗里德曼方程；没有弗里德曼方程，就无法描述宇宙演化；没有宇宙演化，就无法讨论暴胀、扰动、CMB。

**整个逻辑链**:
```
哥白尼原理(均匀+各向同性)
    → FLRW度规(时空几何)
    → 爱因斯坦场方程
    → 弗里德曼方程(宇宙动力学)
    → 尺度因子a(t)的时间演化
    → 红移、距离、年龄等观测预言
```

## 1. 从对称性到度规: FLRW度规是怎么想到的？

### 物理直觉

**核心问题**: 宇宙在大尺度上看起来 everywhere 差不多（没有中心，没有特殊方向）。如何用数学描述？

**思考路径**:

**Step 1**: 把时空分成"时间"和"空间"

我们写下一般时空度规的线元:

$$ds^2 = g_{\mu\nu} dx^\mu dx^\nu$$

我们需要一个分割：时间坐标 $t$ 加上空间坐标 $x^i$（$i=1,2,3$）。

**为什么可以这样分？** 因为宇宙整体均匀——每个观测者看到的情况一样，所以存在一个"共动观测者"的概念，他们感受不到任何本动（peculiar motion）。他们的固有时就是宇宙时 $t$。

**动机**: 如果宇宙各向同性，那么存在一类观测者，他们对所有人来说看到的天空都一样——这就是共动观测者。他们携带的时钟测量的是宇宙年龄。这种分割称为"协变时间切片"（covariant time slicing）。

**Step 2**: 空间部分的最大对称性

三维空间如果是最大对称的（均匀+各向同性），那么它的曲率必须是常数。

**回忆微分几何**：常曲率三维空间的分类。一个三维黎曼流形，如果它具有最大对称性（6个 Killing 矢量：3个平移+3个旋转），那么它的 Riemann 曲率张量只能是:

$$R_{ijkl} = K (g_{ik}g_{jl} - g_{il}g_{jk})$$

其中 $K$ 是一个常数，称为**高斯曲率**（constant sectional curvature）。

对常曲率空间的坐标参数化（见 Misner-Thorne-Wheeler §22.6 或任意微分几何教材），三维线元必可写为:

$$d\sigma^2 = \frac{dr^2}{1-Kr^2} + r^2(d\theta^2 + \sin^2\theta\, d\phi^2)$$

**为什么是这个形式？** 这是三维常曲率空间在"球面坐标"下的标准参数化。具体推导思路是：先由对称性知道空间必须是旋转不变的，所以角部分一定是 $r^2 d\Omega^2$；径向部分由于曲率常数的存在，必须有一个修正因子 $(1-Kr^2)^{-1}$。

其中:
- $K > 0$: 正曲率，三维球面 $S^3$，几何上等价于三维球面 $S^3$
- $K = 0$: 零曲率，平直欧氏空间 $\mathbb{R}^3$
- $K < 0$: 负曲率，双曲空间 $H^3$

尺度因子 $a(t)$ 可以随时间变化（但空间各点相同 → 均匀性）。

**动机**: 常曲率空间的分类是标准微分几何结果。物理学家直接套用。物理上，$K$ 的正负对应闭合/平坦/开放的宇宙几何。

**Step 3**: 时间-空间交叉项为零

如果存在空间各向同性，那么不能有 $t$-$r$ 交叉项（否则某个方向特殊）。

所以 $g_{0i} = 0$（$i = 1,2,3$）。

**动机**: 各向同性禁止任何方向性——包括时间方向和径向之间的关联。

**Step 4**: 时间部分

$$g_{00} = -1$$

（因为我们用共动观测者的固有时，即宇宙时。也可以写为 $g_{00} = -1$，度规符号采用 $(-,+,+,+)$）

**动机**: 共动观测者的世界线是类时测地线，其固有时等于坐标时。这相当于规范选择。

### 结果 — FLRW 度规

将以上四步组合:

$$ds^2 = -dt^2 + a^2(t) \left[ \frac{dr^2}{1-Kr^2} + r^2 (d\theta^2 + \sin^2\theta\, d\phi^2) \right] \tag{1}$$

这就是**FLRW度规**（Friedmann-Lemaître-Robertson-Walker）。

更紧凑地记三维空间部分为 $d\sigma_K^2$，记 $k = K/|K|$ 为归一化曲率 ($k = 0,\pm 1$)，写为:

$$ds^2 = -dt^2 + a^2(t) \left[ \frac{dr^2}{1-kr^2} + r^2 d\Omega^2 \right]$$

### 为什么这样想？

- **不是瞎猜出来的**——是从对称性原理严格推导出来的。
- 类比：电磁学中从 $U(1)$ 规范对称性 → 唯一可能的拉氏量是 $F_{\mu\nu}F^{\mu\nu}$。
- FLRW 是从"宇宙学原理"（宇宙在大尺度上均匀+各向同性）这个对称性假设 → 唯一可能的时空几何。

**物理图像**: 想象一个橡皮气球，表面画满点。气球膨胀 → 点之间距离增加。但每个点看到的邻居分布是一样的（均匀）。这就是 $K=0$ 或 $K=+1$ 的情况。$a(t)$ 就是气球的"半径"。

### 另一种常用坐标：共形时间

引入**共形时间** $\eta$，定义为:

$$d\eta = \frac{dt}{a(t)} \quad\Rightarrow\quad \eta = \int_0^t \frac{dt'}{a(t')}$$

则度规变为:

$$ds^2 = a^2(\eta) \left[ -d\eta^2 + \frac{dr^2}{1-kr^2} + r^2 d\Omega^2 \right] \tag{2}$$

**为什么引入共形时间？** 因为此时度规中的 $-d\eta^2 + d\sigma_k^2$ 部分在光锥结构上与 Minkowski 时空相同。光子的世界线在 $(\eta,r)$ 平面上就是 $45^\circ$ 线（$ds^2=0 \Rightarrow d\eta = \pm dr$）。这对讨论因果结构（如 horizon 问题）非常重要。

## 2. 从爱因斯坦方程到弗里德曼方程

### 物理直觉

**核心问题**: 知道了几何（FLRW度规），怎么知道物质怎么演化？

**思考路径**:

**Step 1**: 爱因斯坦场方程

$$G_{\mu\nu} + \Lambda g_{\mu\nu} = 8\pi G\, T_{\mu\nu} \tag{3}$$

左边 = 几何（爱因斯坦张量 $G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}R g_{\mu\nu}$）
右边 = 物质（应力-能量张量 $T_{\mu\nu}$）

**动机**: 这是广义相对论的基本方程。几何 = 物质。这是已知的物理定律，我们把它应用到 FLRW 度规上。

**Step 2**: 建模物质

宇宙中的物质近似为完美流体:

$$T_{\mu\nu} = (\rho + p) u_\mu u_\nu + p g_{\mu\nu} \tag{4}$$

其中:
- $\rho(t)$ = 能量密度（只依赖时间 → 均匀性）
- $p(t)$ = 压强
- $u^\mu = (1, 0, 0, 0)$（共动观测者，静止在坐标中）

**动机**: 大尺度上宇宙看起来像流体。没有粘滞、没有热传导 → 完美流体近似。注意 $\rho$ 和 $p$ 只能是 $t$ 的函数（因为空间均匀性）。

**Step 3**: 计算 Einstein 张量

这一步是纯计算。代入 FLRW 度规，计算:

**Christoffel 符号**:
$$\Gamma^\lambda_{\mu\nu} = \frac{1}{2}g^{\lambda\sigma}(\partial_\mu g_{\sigma\nu} + \partial_\nu g_{\sigma\mu} - \partial_\sigma g_{\mu\nu})$$

对 FLRW 度规，非零 Christoffel 符号为:
$$\Gamma^0_{ij} = a\dot{a}\, \tilde{g}_{ij}, \quad \Gamma^i_{0j} = \frac{\dot{a}}{a}\delta^i_j, \quad \Gamma^i_{jk} = \tilde{\Gamma}^i_{jk}$$

其中 $\tilde{g}_{ij}$ 是三维常曲率空间的度规，$\tilde{\Gamma}^i_{jk}$ 是其 Christoffel 符号。

**动机**: 这纯粹是代入计算。关键点在于 $\dot{a}/a = H$ 的出现——Hubble 参数自然地从度规的时间导数中诞生。

**Riemann 张量 → Ricci 张量 → Ricci 标量 → Einstein 张量**:

关键结果（代入后算出的分量）:

$$G_{00} = 3\left[\left(\frac{\dot{a}}{a}\right)^2 + \frac{K}{a^2}\right] = 3\left[H^2 + \frac{K}{a^2}\right] \tag{5}$$

$$G_{ij} = -\left(2\frac{\ddot{a}}{a} + \frac{\dot{a}^2}{a^2} + \frac{K}{a^2}\right) g_{ij} \tag{6}$$

**动机**: 这不是"想到"的，是"算出来"的。学生需要知道的是**为什么代进去就能得到结果**，而不是记住计算细节。考试时公式会给出，所以关键是理解物理含义。

**Step 4**: 写出 $00$ 分量和 $ii$ 分量方程

**$00$ 分量**（第一个弗里德曼方程）:

$$G_{00} + \Lambda g_{00} = 8\pi G\, T_{00}$$

$$3\left[H^2 + \frac{K}{a^2}\right] + \Lambda = 8\pi G\, \rho$$

整理得:

$$H^2 = \frac{8\pi G}{3}\rho - \frac{K}{a^2} + \frac{\Lambda}{3} \tag{7}$$

这就是**第一个弗里德曼方程**。

**物理含义**:
- $\rho$ 大 → $H$ 大（物质能量密度高 → 膨胀速率大，因为早期宇宙中物质多时膨胀也快）
- $K > 0$ → $H$ 小（正曲率"减速"膨胀，几何上球面"卷回去"）
- $\Lambda > 0$ → $H$ 大（暗能量驱动加速膨胀）

**$ii$ 分量**（第二个弗里德曼方程，加速度方程）:

$$G_{ij} + \Lambda g_{ij} = 8\pi G\, T_{ij}$$

$$-\left(2\frac{\ddot{a}}{a} + \frac{\dot{a}^2}{a^2} + \frac{K}{a^2}\right) + \Lambda = -8\pi G\, p$$

利用方程 (7) 消去 $\frac{\dot{a}^2}{a^2} + \frac{K}{a^2} = H^2$:

$$2\frac{\ddot{a}}{a} = -H^2 - \frac{K}{a^2} + \frac{8\pi G}{3}\rho + \Lambda - H^2 - \frac{K}{a^2} + \Lambda$$

$$\frac{\ddot{a}}{a} = -\frac{4\pi G}{3}(\rho + 3p) + \frac{\Lambda}{3} \tag{8}$$

这就是**第二个弗里德曼方程**（加速度方程）。

**物理含义**: 加速度由 $(\rho + 3p)$ 决定！
- $\rho > 0, p \geq 0 \Rightarrow \ddot{a} < 0$（引力是吸引的，减速膨胀）
- 但 $p < -\rho/3$ 时（如暗能量或暴胀子势能）→ $\ddot{a} > 0$（加速膨胀）

### 为什么 $(\rho + 3p)$ 出现？

这是广义相对论和牛顿引力的关键区别。在 GR 中，**压强也产生引力**。

- **辐射**: $p = \rho/3 \Rightarrow (\rho + 3p) = 2\rho$ → 引力效果是物质的 2 倍
- **物质**: $p \approx 0 \Rightarrow (\rho + 3p) \approx \rho$
- **暗能量**: $p \approx -\rho \Rightarrow (\rho + 3p) \approx -2\rho$ → 反引力（加速）

**动机**: 在牛顿理论中，引力只由质量密度决定。在 GR 中，完整的应力-能量张量 $T_{\mu\nu}$ 都参与引力相互作用。特别是，压强出现在 $T_{ii}$ 中，通过 Einstein 方程影响时空曲率。$\rho + 3p$ 正是有效引力源的体现。

### 能量守恒方程

从 $\nabla_\mu T^{\mu\nu} = 0$ 自动得到:

$$\dot{\rho} + 3H(\rho + p) = 0 \tag{9}$$

**详细推导**: 利用 $\nabla_\mu T^{\mu 0} = 0$:

$$\nabla_\mu T^{\mu 0} = \partial_\mu T^{\mu 0} + \Gamma^\mu_{\mu\lambda} T^{\lambda 0} + \Gamma^0_{\mu\lambda} T^{\mu\lambda} = 0$$

对于完美流体 $T^{\mu\nu} = (\rho+p)u^\mu u^\nu + p g^{\mu\nu}$，在共动坐标系中 $u^\mu = (1,0,0,0)$:

$$T^{00} = \rho, \quad T^{ij} = p g^{ij}$$

计算各 Christoffel 符号的贡献，最终得到:

$$\dot{\rho} + 3\frac{\dot{a}}{a}(\rho + p) = 0$$

**物理含义**:
- $3H\rho$: 体积膨胀导致密度稀释（类似绝热膨胀）
- $3Hp$: 压强做功导致的能量变化

**记忆方法**: "$3H$" 来自三维空间的膨胀速率 $\nabla_i v^i = 3H$（$\nabla \cdot \vec{v} = \nabla \cdot (H\vec{r}) = 3H$）。

### 流体力学方程的物理图像

方程 (9) 其实就是热力学中的**绝热膨胀关系**。考虑体积 $V \propto a^3$ 内的总能量 $E = \rho V$:

$$dE = -p\, dV \quad\Rightarrow\quad \frac{d}{dt}(\rho a^3) = -p \frac{d}{dt}(a^3)$$

$$\dot{\rho} a^3 + 3\rho a^2 \dot{a} = -3p a^2 \dot{a}$$

除以 $a^3$ 即得 $\dot{\rho} + 3H(\rho + p) = 0$。

**动机**: 这是第一定律 $dE = -p\, dV$ 在宇宙学尺度上的直接应用。宇宙膨胀时，流体体积增大，对外做功，能量减少。

## 3. 各成分密度的演化

### 从能量守恒推导

$$\dot{\rho} + 3H(\rho + p) = 0$$

对于每种成分，有**状态方程** $p = w\rho$，其中 $w$ 是状态方程参数:

代入:

$$\dot{\rho} + 3H\rho(1 + w) = 0$$

注意到 $H = \dot{a}/a$，即 $H dt = da/a$:

$$\frac{d\rho}{\rho} = -3(1+w) \frac{da}{a}$$

积分:

$$\ln \rho = -3(1+w) \ln a + \text{const}$$

$$\rho \propto a^{-3(1+w)} \tag{10}$$

对不同成分:

**(1) 物质（非相对论）**: $w = 0$

$$\rho_m \propto a^{-3}$$

**物理原因**: 体积增大 → 粒子数密度稀释。假设粒子数守恒，则 $\rho_m = m n_m \propto a^{-3}$。

**(2) 辐射（相对论粒子，如光子、中微子）**: $w = 1/3$

$$\rho_r \propto a^{-4}$$

**物理原因**: 除了体积稀释 ($a^{-3}$)，还有**红移**效应——每个光子的能量 $E = h\nu \propto 1/a$。所以额外的 $a^{-1}$。总效应: $\rho_r \propto a^{-3} \times a^{-1} = a^{-4}$。

**(3) 宇宙学常数（暗能量）**: $w = -1$

$$\rho_\Lambda = \text{const}$$

**物理原因**: 它是真空能，不随体积稀释。体积增大，总能量增大 → 密度不变。

**(4) 一般标量场**（将在模块2讨论）: 有效 $w$ 可以在 $-1$ 到 $1$ 之间变化。

### 为什么这样依赖？

这些关系可以从粒子物理直觉理解:
- 物质的粒子能量不随膨胀改变（静止质量不变）→ 仅稀释
- 辐射粒子能量随 $1/a$ 降低（波长被拉长）→ 额外稀释
- 真空能是时空的固有属性 → 不变

### 临界密度与密度参数

**定义**: 

$$\rho_c \equiv \frac{3H^2}{8\pi G} \tag{11}$$

**为什么这样定义？** 因为如果 $K = 0$（平坦宇宙），从弗里德曼方程 (7)（设 $\Lambda=0$ 暂且不论）:

$$H^2 = \frac{8\pi G}{3}\rho \quad\Rightarrow\quad \rho = \frac{3H^2}{8\pi G} = \rho_c$$

所以 $\rho_c$ 是平坦宇宙所需的密度。

**定义密度参数**:

$$\Omega \equiv \frac{\rho}{\rho_c}, \quad \Omega_K \equiv -\frac{K}{a^2H^2} \tag{12}$$

将方程 (7) 两边除以 $H^2$:

$$1 = \frac{8\pi G}{3H^2}\rho - \frac{K}{a^2H^2} + \frac{\Lambda}{3H^2}$$

$$1 = \Omega + \Omega_K + \Omega_\Lambda \tag{13}$$

其中 $\Omega_\Lambda \equiv \frac{\Lambda}{3H^2}$。

**观测**: 现代宇宙学的精确测量表明:
$$\Omega_{\text{total}} \approx 1$$

这意味着宇宙几乎是平坦的。具体地:
$$\Omega_m \approx 0.3, \quad \Omega_\Lambda \approx 0.7, \quad \Omega_K \approx 0$$

**动机**: 这个关系将宇宙的几何（$K$）与物质含量（$\Omega$）直接联系了起来。平坦宇宙是暴胀理论的重要预言之一。

### 能量密度随红移的演化

用红移 $z = a_0/a - 1$（$a_0 = 1$ 为今天的尺度因子）来写:

$$\rho(z) = \rho_0(1+z)^{3(1+w)}$$

具体地:
- $\rho_m(z) = \rho_{m0}(1+z)^3$
- $\rho_r(z) = \rho_{r0}(1+z)^4$
- $\rho_\Lambda(z) = \rho_{\Lambda0} = \text{const}$

由此:

$$H^2(z) = H_0^2\left[ \Omega_{r0}(1+z)^4 + \Omega_{m0}(1+z)^3 + \Omega_{\Lambda0} + \Omega_{K0}(1+z)^2 \right] \tag{14}$$

其中 $\Omega_{K0} = -K/(a_0^2 H_0^2)$。

### 物质-辐射等时的红移

等时条件: $\rho_m(a_{eq}) = \rho_r(a_{eq})$

$$\rho_{m0} a_{eq}^{-3} = \rho_{r0} a_{eq}^{-4}$$

$$a_{eq} = \frac{\rho_{r0}}{\rho_{m0}} = \frac{\Omega_{r0}}{\Omega_{m0}}$$

$$1 + z_{eq} = \frac{\Omega_{m0}}{\Omega_{r0}} \approx \frac{0.3}{9 \times 10^{-5}} \approx 3400 \tag{15}$$

其中 $\Omega_{r0} \approx 9 \times 10^{-5}$（包括光子和中微子）。

**物理意义**: $z \gg 3400$ 时宇宙由辐射主导；$z \ll 3400$ 时宇宙由物质主导。等时附近是结构形成的关键时期。

## 4. 这个模块怎么考？

老师明确说："公式会给出，考代入计算"。

**典型考法**:

**1. "给定 $\rho = \rho_m + \rho_r$，求 $H(z)$"**

→ 用 $\rho_m = \rho_{m0}(1+z)^3, \rho_r = \rho_{r0}(1+z)^4$

$$H^2(z) = H_0^2\left[\Omega_{r0}(1+z)^4 + \Omega_{m0}(1+z)^3 + \Omega_{\Lambda0}\right]$$

**2. "物质-辐射相等时的红移"**

$$z_{eq} \approx 3400$$

直接代入 $\Omega_{m0}$ 和 $\Omega_{r0}$ 即可。

**3. "给定 $\Omega_m, \Omega_\Lambda$，求宇宙年龄"**

$$t_0 = \int_0^\infty \frac{dz}{(1+z)H(z)}$$

**4. "判断哪种成分在何时主导"**

比较各 $\Omega_i(1+z)^{3(1+w_i)}$ 随 $z$ 的变化。高 $z$ 时 $w$ 小的成分主导（因为指数大）。

## 5. 下一步：从引力到暴胀

-module1 提供了宇宙的动力学框架：知道了几何（FLRW）和演化方程（弗里德曼方程），我们可以描述任何物质组成的宇宙的膨胀历史。

但是，标准热大爆炸模型有几个严重的疑难（视界问题、平坦性问题、单极问题）。要解决这些问题，我们需要一个**驱动加速膨胀的物理源**。

从方程 (8) 看，$\ddot{a} > 0$ 的条件是 $\rho + 3p < 0$，即 $p < -\rho/3$。在标准物质或辐射中，这不可能满足。

**所以我们需要的是一种具有负压的物质形态**。自然地，我们转向**标量场**——它在粒子物理中已有先例（如 Higgs 场），且在弯曲时空中可以产生有效的负压。这就是下一个模块的内容。
