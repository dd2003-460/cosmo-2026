## 1. 暴胀解决的三个疑难 — 每个怎么想到的？

### 1.1 视界问题

**问题**: CMB在天空相反两点温度相同（精度 1/10⁵）。但这两点在今天的光锥中没有因果联系 — 光从大爆炸以来还没足够的时间让它们交换信息。

**怎么想到的**: 计算共动 horizon:
```
共动 horizon: d_H(t) = ∫₀^t c dt'/a(t')

两点因果联系的充要条件: 它们的共动距离 < d_H(t)

CMB 光子最后散射时(z*≈1100)，共动 horizon ≈ 0.1 Mpc (共动)
而天空中相隔 1° 的两点 → 共动距离 ≈ 100 Mpc

100 >> 0.1 → 没有因果联系 → 但温度相同 → 疑难
```

**暴胀怎么解决**:
```
暴胀期间: a(t) ∝ e^{Ht}

共动 horizon 在暴胀期间变小！因为 ∫ dt/a(t) 收敛。

物理过程:
1. 暴胀前: 可观测宇宙来自一个很小的因果连通区域
2. 暴胀: 这个区域被指数拉伸 → 远大于今天的 horizon
3. 暴胀后: 热大爆炸演化 → 到今天

结论: 现在看到的整个 CMB 天区，暴胀前是因果连通的 ✓
```

### 1.2 平坦性问题

**问题**: 今天观测到 Ω ≈ 1 (平坦宇宙)。但:
```
从弗里德曼方程: 1 - Ω = K/(a²H²)

物质主导: a ∝ t^{2/3}, H = 2/(3t) → a²H² ∝ t^{−2/3} → |1-Ω| ∝ t^{2/3} (增长)
辐射主导: a ∝ t^{1/2}, H = 1/(2t) → a²H² ∝ t⁰ → |1-Ω| ∝ t (增长)

这意味着: 如果今天 |1-Ω| ≈ 0.01，那么在 Planck 时代 |1-Ω| ≈ 10⁻⁶⁰

为什么初始条件这么精确？
```

**暴胀怎么解决**:
```
暴胀: a ∝ e^{Ht} → a²H² 指数增长

|1-Ω| = K/(a²H²) → 暴胀期间指数减小 → 趋近于 0

不管初始 Ω 偏离 1 多少，暴胀后 Ω 极其接近 1。

类比: 吹一个大气球，表面看起来越来越平。
```

### 1.3 磁单极问题

**问题**: Grand Unified Theory (GUT) 相变应该产生大量磁单极。但从未观测到。

**暴胀怎么解决**: 暴胀稀释了单极密度。暴胀前存在的单极被稀释到可观测宇宙中几乎为零。

## 2. 标量场在 FRW 背景下的运动方程

### 物理直觉

**核心问题**: 需要一个场，它具有负压，能驱动指数膨胀。为什么选标量场？

**思考路径**:

**Step 1**: 为什么是标量场？
```
标量场 φ(x) 是 simplest dynamical field.
- 矢量场有方向性 → 破坏各向同性
- 张量场 → 引力子（已经是几何的一部分）
- 标量场 → 各向同性，不破坏对称性 ✓

而且，标量场的拉氏量形式简单:
L = -½∂_μφ∂^μφ - V(φ)
```

**动机**: 在粒子物理中，标量场已经存在 — Higgs 场就是标量场。用它驱动暴胀是自然的推广。

**Step 2**: FRW 背景下的运动方程
```
一般 curved spacetime 中标量场的运动方程:
□φ - V'(φ) = 0

其中 □ = (1/√-g)∂_μ(√-g g^μν ∂_ν) 是 covariant d'Alembertian

对 FLRW 度规:
ds² = -dt² + a²(t)dx²
√-g = a³(t)

□φ = (1/a³)∂_t(a³∂_tφ) + (1/a²)∇²φ
   = φ̈ + 3Hφ̇ - (1/a²)∇²φ

所以运动方程:
φ̈ + 3Hφ̇ + V'(φ) - (1/a²)∇²φ = 0

对于均匀背景 φ = φ(t): ∇²φ = 0

→ φ̈ + 3Hφ̇ + V'(φ) = 0    (标量场运动方程)
```

**动机**: 这是弯曲时空中标量场方程的直接应用。物理学家知道 GR 中的场方程，直接代入 FLRW 度规。

**Step 3**: 每一项的物理含义
```
φ̈ : 场的加速度
3Hφ̇ : "Hubble 摩擦"项 — 宇宙膨胀就像在黏性介质中运动
V'(φ) : 势梯度 — 驱动力

类比: 一个小球在势 V(φ) 中滚动，同时受到与速度成正比的摩擦。
摩擦系数 = 3H。H 越大，摩擦越大。
```

**为什么叫"Hubble 摩擦"？**
```
想象一个球在山上滚动。如果没有摩擦(3H=0)，球会振荡。
如果有大摩擦(3H >> 球的质量)，球缓慢滚动(overdamped)。

暴胀的条件就是"大摩擦" → 慢滚近似。
```

**Step 4**: 能量密度和压强
```
从应力-能量张量 T_μν = ∂_μφ∂_νφ - g_μν[½∂_αφ∂^αφ + V(φ)]

均匀背景 φ(t):

ρ_φ = T_00 = ½φ̇² + V(φ)     (动能 + 势能)
p_φ = T_ii/3 = ½φ̇² - V(φ)    (动能 - 势能)

关键: 如果 V(φ) >> ½φ̇² (势能主导):
p_φ ≈ -V(φ) < 0 → 负压 ✓

负压 → 从弗里德曼方程 #2: ä/a = -(4πG/3)(ρ + 3p)
如果 p < -ρ/3 → ä > 0 → 加速膨胀 ✓
```

### 为什么 V >> ½φ̇² 就能加速？

```
ρ + 3p = (½φ̇² + V) + 3(½φ̇² - V) = 2φ̇² - 2V

如果 V >> ½φ̇²: ρ + 3p ≈ -2V < 0 → ä > 0

物理直觉: 势能产生负压，负压产生反引力(加速膨胀)。
这和牛顿引力完全不同 — 在 GR 中，压强也产生引力。
```

## 3. 慢滚近似

### 从"大摩擦"到近似方程

**思考路径**:

**Step 1**: 慢滚条件
```
类比中，overdamped 的条件是摩擦项 >> 加速度项:

3H|φ̇| >> |φ̈|

同时，势能主导: V >> ½φ̇²

这两个条件合起来就是"慢滚条件"。
```

**Step 2**: 慢滚近似下的运动方程
```
原方程: φ̈ + 3Hφ̇ + V' = 0

慢滚近似: φ̈ << 3Hφ̇, V' << V (势能变化缓慢)

→ 3Hφ̇ ≈ -V'(φ)    (慢滚运动方程 #1)

同时，能量密度:
ρ ≈ V(φ)

从弗里德曼方程:
3H² ≈ 8πG V(φ)/3

→ H² ≈ V/(3M_pl²)    (慢滚运动方程 #2)

其中 M_pl = (8πG)⁻¹/² = 约 2.4 × 10¹⁸ GeV
```

**动机**: 把两个方程联立，消去 φ̇，得到 φ 和 V 的直接关系:
```
φ̇ ≈ -V'/(3H) ≈ -V'/(√(3) × √(8πG/3) × √V) = -V'/(3M_pl√V)
```

**Step 3**: 定义慢滚参数
```
为了量化"慢滚"，定义无量纲参数:

ε_V = M_pl²/2 × (V'/V)²
η_V = M_pl² × V''/V

为什么这样定义?

ε_V 衡量势能斜率的平方:
- V'/V 有量纲 [mass]⁻¹
- 乘以 M_pl² 得到无量纲数
- ε_V << 1 等价于 |V'| << V/M_pl ✓

η_V 衡量势能曲率:
- V''/V 有量纲 [mass]²
- 乘以 M_pl² 得到无量纲数
- |η_V| << 1 等价于 |V''| << V/M_pl² ✓
```

**Step 4**: e-folding 数
```
N = ∫ H dt = ln(a_end/a)

为什么用 N? 因为暴胀需要至少 N ≈ 60 e-foldings 才能解决疑难。

N 的计算:
N = ∫_{t_exit}^{t_end} H dt = ∫_{φ_exit}^{φ_end} (H/φ̇) dφ

用慢滚近似:
H/φ̇ = H/(-V'/3H) = -3H²/V' = -1/(3M_pl² V'/V)

→ N ≈ ∫_{φ_end}^{φ_exit} (V/V') dφ / M_pl²    (用 N 表示暴胀)
```

**动机**: 把时间积分转换为对场的积分 — 因为 φ 是单调变化的，这是一个合法的变量替换。
